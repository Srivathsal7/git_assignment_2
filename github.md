GitHub Features



1\. Repository hosting – stores project code online.

2\. Version control – tracks changes in code.

3\. Branching – allows working on new features separately.

4\. Pull Requests – used to merge code changes.

5\. Issues – track bugs and tasks.

6\. Actions – CI/CD automation.

7\. Collaboration – multiple developers can work together.

8\. Releases – publish versions of software.
